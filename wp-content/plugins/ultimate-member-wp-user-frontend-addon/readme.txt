Integration of "Ultimate Member" + "WP User Fronted" in user profiles

== Description ==
This plugin integrates “Ultimate Member” with “WP User Fronted”. It adds to the user profile additional pages to “add posts” and “list of posts” (with edit link). 
You can select what roles you want to show the additional pages to.

== Installation ==

1. Install Ultimate Member
2. Install WP User Fronted
3. Set "Edit page" for WP User Fronted
4. Install "Ultimate Member + WP User Frontend"
4.1. Select roles where you want to display additional tabs
4.2. Fill Form ID in settings page from Wp User Frontend

== Frequently Asked Questions ==

= WP User Frontend Pro - Support? =

Our version was tested only on free version of plugin.

= "My Post" tab display useless things =

You can set things to display at User Frontend Settings.

= Tabs dosen't display =

You need to set Roles and Form ID in "UFA Settings" page.

= Form ID ? =

After creating form at User Frontend you got Form ID. You must specify the form id!
